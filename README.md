# conversion-decimal-binario
El proyecto está configurado para ejecutarse automáticamente en GitHub Codespaces mediante un contenedor de desarrollo (.devcontainer) que contiene todo lo necesario para correr programas en Python 3.10.
Esto garantiza que cualquier persona pueda abrir el repositorio en Codespaces y ejecutar el código sin instalar nada adicional.

Pasos para ingresar al codespace
Entra al repositorio en GitHub.
Clic en el botón verde “Code” → “Create codespace on main”.
Espera a que GitHub cree el entorno.
Una vez abierto el editor de Codespaces:
Abre la terminal desde la barra superior: Terminal - New Terminal
Ingresa un número decimal de dos cifras
El programa mostrará un número binario de 8 bits
